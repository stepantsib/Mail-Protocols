# Mail Protocols Clients (SMTP, IMAP, POP3)

Данный репозиторий содержит набор консольных скриптов на Python для работы с электронной почтой по базовым сетевым протоколам: SMTP, IMAP и POP3. Клиенты поддерживают безопасное соединение (SSL), авторизацию и гибкую настройку через аргументы командной строки.

## 🚀 Основные возможности
* **SMTP (`smtp.py`)**: Отправка электронных писем, поддержка прикрепления файлов (вложений) из указанной директории.
* **IMAP (`imap.py`)**: Подключение к почтовому ящику, чтение писем, поддержка выборки сообщений по заданному диапазону.
* **POP3 (`pop3.py`)**: Загрузка и чтение входящих сообщений с поддержкой указания диапазона.

## 🛠 Общие аргументы командной строки
Большинство скриптов используют общий синтаксис флагов для настройки подключения:
* `-s` : Адрес сервера и порт (например, `smtp.gmail.com:465`)
* `-u` : Email пользователя для авторизации (логин)
* `-f` : Email отправителя (для SMTP)
* `-t` : Email получателя (для SMTP)
* `-d` : Директория с файлами для вложений
* `-n` : Диапазон номеров писем для чтения (например, `-n 1 5`)
* `--ssl` : Использовать защищенное соединение (SSL/TLS)
* `--auth`: Выполнить авторизацию на сервере
* `-v` : Verbose-режим (подробный вывод логов взаимодействия с сервером)

## 📖 Примеры использования

> **Важно:** При запуске скриптов с параметром авторизации программа запросит ввод пароля. Необходимо использовать **пароли приложений** (App Passwords), а не основной пароль от почтового ящика.

### 1. SMTP-клиент (Отправка писем)
Отправка письма с вложениями из папки `pictures` через сервер Mail.ru с подробным логированием:
```text
    python smtp.py -s smtp.mail.ru:465 -f sender@mail.ru -t receiver@gmail.com -d pictures --ssl --auth -v
```
Отправка через сервер Gmail:
```text
  python smtp.py -s smtp.gmail.com:465 -f sender@gmail.com -t receiver@gmail.com -d pictures --ssl --auth -v
```
### 2. IMAP-клиент (Чтение)

Чтение всех писем (по умолчанию) с сервера Mail.ru:
```text
  python imap.py -s imap.mail.ru:993 -u your_email@mail.ru --ssl -v
```
Чтение диапазона писем (с 10 по 15) с сервера Gmail: 
```text
  python imap.py -s imap.gmail.com:993 -u your_email@gmail.com --ssl -n 10 15 -v
```
Чтение первых 5 писем с сервера Yandex:
```text
  python imap.py -s imap.yandex.ru:993 -u your_email@yandex.ru --ssl -n 1 5 -v
```
### 3. POP3-клиент (Получение писем)

Получение первых 5 писем через сервер Mail.ru:
```text
  python pop3.py -s pop.mail.ru:995 -u your_email@mail.ru --ssl -n 1 5 -v
```
Получение первых 5 писем через сервер Gmail:
```text
  python pop3.py -s pop.gmail.com:995 -u your_email@gmail.com --ssl -n 1 5 -v
```
Получение первых 5 писем через сервер Yandex:
```text
  python pop3.py -s pop.yandex.ru:995 -u your_email@yandex.ru --ssl -n 1 5 -v
```
